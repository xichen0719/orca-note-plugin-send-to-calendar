# Send to Calendar

虎鲸笔记插件：通过块右键菜单，把当前选中的块复制到一个或多个日期的日记中。

## 使用方式

1. 在虎鲸笔记中选中一个或多个块。
2. 右键块菜单，点击“发送到日记”。
3. 在弹窗里选择日期范围：点击开始日期，移动鼠标可预览范围，再点击结束日期确认。
4. 点击“发送”。

插件会把选中的块追加到每天日记的末尾。复制使用虎鲸笔记原生的 `core.editor.copyBlocks`，因此会尽量保留原有块结构和子块；新副本的创建时间由虎鲸笔记生成。

## 行为说明

- 日期范围可以正选，也可以反选，例如 `06.02 -> 06.01` 会按 `06.01 -> 06.02` 处理。
- 日期范围会在日历和底部状态栏中实时预览；发送按钮会按当前显示的范围执行。
- 同一次发送里，重复日期和重复块会自动去重。
- 插件不会扫描历史副本；如果分多次发送到同一天，会按复制操作生成多份副本。
- 一次批量发送会作为一个撤销步骤处理。
- 可在插件设置里开启“发送前创建备份”。开启后，插件会在复制前调用虎鲸内部全库备份入口，备份名格式为 `send-to-calendar-YYMMDD-HHmmss`。该入口来自实际 DevTools 捕获，不属于当前公开文档 API，因此默认关闭。

## 开发与测试

```bash
pnpm install
pnpm exec tsc --noEmit
pnpm build
```

本地测试时，可以把构建后的轻量插件目录放到：

```text
\orca\plugins\send-to-calendar
```
